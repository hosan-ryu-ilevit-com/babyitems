- ⚠ 두꺼운 소재의 옷감에는 일반 다리미만큼의 완벽한 다림질 효과를 기대하기 어려움
- ⚠ 물방울이 튈 수 있음
- ⚠ 일부 제품의 경우 스팀 버튼을 계속 누르고 있어야 함
- ⚠ 스탠드형의 경우 부피가 크고 보관이 번거로울 수 있음
- ⚠ 핸디형의 경우 옷감을 직접 잡고 다려야 하는 불편함이 있을 수 있음
